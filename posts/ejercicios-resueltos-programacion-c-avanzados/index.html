Ejercicio 70 de programación en C: lista doblemente enlazada resuelta

Si estás buscando ejercicios resueltos de programación en C y quieres subir de nivel en estructuras de datos, este ejercicio es una base muy útil.

En este caso implementamos una lista doblemente enlazada: inserción al final, recorrido hacia adelante, recorrido hacia atrás y liberación completa de memoria.

Enunciado

Implementa una estructura de nodo con:

  • valor
  • puntero a siguiente
  • puntero a anterior

Y crea funciones para:

  • insertar al final,
  • recorrer la lista hacia adelante,
  • recorrer la lista hacia atrás (desde la cola),
  • liberar toda la memoria al finalizar.

Solución en C (ejercicio 70 resuelto)

#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int valor;
    struct Nodo *siguiente;
    struct Nodo *anterior;
} Nodo;

Nodo *insertarFinal(Nodo *cabeza, int valor) {
    Nodo *nuevo = (Nodo *)malloc(sizeof(Nodo));
    if (nuevo == NULL) {
        fprintf(stderr, "Error al reservar memoria.\n");
        return cabeza;
    }

    nuevo->valor = valor;
    nuevo->siguiente = NULL;
    nuevo->anterior = NULL;

    if (cabeza == NULL) {
        return nuevo;
    }

    Nodo *actual = cabeza;
    while (actual->siguiente != NULL) {
        actual = actual->siguiente;
    }

    actual->siguiente = nuevo;
    nuevo->anterior = actual;
    return cabeza;
}

void recorrerAdelante(Nodo *cabeza) {
    printf("Recorrido hacia adelante:\n");
    while (cabeza != NULL) {
        printf("%d ", cabeza->valor);
        cabeza = cabeza->siguiente;
    }
    printf("\n");
}

void recorrerAtras(Nodo *cola) {
    printf("Recorrido hacia atrás:\n");
    while (cola != NULL) {
        printf("%d ", cola->valor);
        cola = cola->anterior;
    }
    printf("\n");
}

Nodo *obtenerUltimo(Nodo *cabeza) {
    if (cabeza == NULL) return NULL;
    while (cabeza->siguiente != NULL) {
        cabeza = cabeza->siguiente;
    }
    return cabeza;
}

void liberarLista(Nodo *cabeza) {
    while (cabeza != NULL) {
        Nodo *temp = cabeza;
        cabeza = cabeza->siguiente;
        free(temp);
    }
}

int main(void) {
    Nodo *lista = NULL;

    lista = insertarFinal(lista, 10);
    lista = insertarFinal(lista, 20);
    lista = insertarFinal(lista, 30);

    recorrerAdelante(lista);

    Nodo *cola = obtenerUltimo(lista);
    recorrerAtras(cola);

    liberarLista(lista);
    return 0;
}

Qué se practica con este ejercicio

  • punteros en estructuras,
  • consistencia de enlaces dobles (siguiente y anterior),
  • recorrido bidireccional,
  • gestión de memoria dinámica sin fugas.

Salida esperada

Recorrido hacia adelante:
10 20 30
Recorrido hacia atrás:
30 20 10
