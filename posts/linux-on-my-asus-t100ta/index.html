Linux en mi Asus T100TA

Rodrigo Iglesias
 2015-09-07  685 palabras   4 minutos 

Esta entrada va a ser mi diario de Linux, para poder mantener un registro de lo que hago en la tablet. Bueno, empecemos desde el principio:

Windows me odia

Este cambio empezó con la nueva actualización del SO de Microsoft, Windows 10. Quería probarlo en mi tablet, una Asus Transformer T100, para ver cómo funcionaba. Aunque leí mucho sobre su falta de privacidad, también tenía curiosidad por todos los cambios como el soporte multi-escritorio (que tanto Linux como Macintosh tienen desde hace años) y Cortana, que ofrecía muchas características que parecían útiles.

Así que… lo instalé y, como puedes imaginar, duró unos 5 días instalado. Encontré cada cosa nueva un desperdicio de memoria y CPU, y tuve que reinstalar algunos drivers como el touchpad, porque no funcionaba correctamente. Afortunadamente, Microsoft ofrecía una degradación a Windows 8.1 si no te gustaba Windows 10, así que lo intenté. Todo fue perfecto, excepto por una cosa: Ese nuevo driver que se instaló durante el tiempo que estuve probando Windows 10 no se degradó o algo así, y el escritorio seguía parpadeando y reiniciándose incluso si reiniciaba el SO. Ese fue el momento en que reuní fuerzas para instalar Ubuntu en mi tablet.

¿Por qué Ubuntu? Porque es el único Linux bien ajustado para tener casi todos los componentes funcionando. Hay algunas excepciones de partes que no funcionan, como la cámara, el micrófono y los botones físicos (GPIO), pero el resto funciona de manera realmente estable.

El inicio del viaje

Teniendo esto en cuenta, descargué Asus T100-TA Magic Stick ISO, versión 1.5, arranqué Ubuntu y lo instalé con la herramienta incluida para hacerlo. Sobrescribí completamente la memoria flash interna, incluida la partición UEFI. Todavía tengo una copia de seguridad en mi disco duro externo por si acaso quiero recuperar mi sistema como estaba hace un par de meses, así que no tengo miedo de hacerlo.

Lo primero que hice con el nuevo SO fue eliminar Unity. Odio ese entorno de escritorio, así que instalé XFCE. Para hacerlo, usé una herramienta llamada tasksel. Usé los siguientes comandos para instalarlo y ejecutarlo:

~ $ sudo apt-get install tasksel
~ $ sudo tasksel

Después de esto, marqué Xubuntu-desktop y desmarqué Ubuntu-desktop. Tienes que esperar un par de minutos para descargar XFCE y desinstalar Unity y otras aplicaciones relacionadas, y luego el SO se reinicia y tienes Xubuntu totalmente compatible con tu tablet.

Programas y más programas

Luego, actualicé el SO e instalé algunos programas útiles:

~ $ sudo apt-get update && sudo apt-get upgrade
~ $ sudo apt-get install libreoffice gdebi-core synaptic aptitude vlc xubuntu-restricted-extras libavcodec-extra rar wireshark

Además, instalé Dropbox y Google Chrome (realmente no me gusta Chromium ahora, porque Google ha hecho un gran avance en manejo de memoria y Chromium todavía es un poco más lento y más grande que Chrome).

Para instalar Chrome usé:

~ $ wget https://dl.google.com/linux/direct/google-chrome-stable_current_i386.deb
~ $ sudo gdebi google-chrome-stable_current_i386.deb

Para instalar Dropbox, fui a dropbox.com/install y descargué el archivo *.deb, y seguí las instrucciones.

Y eso es todo lo que hice hoy. Espero que el soporte del touchpad se vuelva mejor y nos dé multi-touch como lo hace Windows. Esa es la única cosa que extraño, la capacidad de desplazar páginas con el touchpad.

Para más información, echa un vistazo a nuestro Grupo de Google+ (nota: Google+ cerró en 2019, puedes ver el grupo archivado aquí. Ahora la comunidad está en Facebook).

(Actualización de junio de 2018)

Aunque estaba muy motivado con el cambio a Linux, volví a Windows después de 6 meses usando Linux por las siguientes razones:

  1. Mi sensación del rendimiento no era la misma que usando Windows. Incluso en 2018, no tenemos soporte completo para esta tablet, y tenemos que tomar algunos atajos para tener todo funcionando. Y hay algunas cosas que probablemente no funcionen fácilmente, como la cámara.
  2. No estaba usando Linux para nada que no pudiera hacer en Windows y necesitaba algunos programas de Windows para mis cursos universitarios.

Sin embargo, tengo la sensación constante de que tal vez debería intentarlo de nuevo. ¿Lo haré? Quién sabe, pero si lo hago, te mantendré informado aquí ;)
