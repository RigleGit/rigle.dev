3dslibris

Lector de ebooks homebrew para Nintendo 3DS. Soporta EPUB, FB2, MOBI, PDF, CBZ y XPS.

Lector de ebooks y manga homebrew para Nintendo 3DS, basado en el proyecto original de Nintendo DS dslibris.

Características

  • Biblioteca con cuadrícula y vista de lista, portadas, metadatos y navegación táctil.
  • Seis temas de lectura con pantallas de inicio y gradientes de lector.
  • Formatos: EPUB, FB2, TXT, RTF, ODT, MOBI, PDF, CBZ y XPS.
  • EPUB con reflow, TOC, marcadores, ir a página, paginación cacheada y soporte de formato inline/bloque.
  • Visor de layout fijo para manga y documentos (CBZ, PDF, XPS) con zoom, paneo, navegación por índice y previsualización de página completa.
  • Fuentes de respaldo incluidas para mayor cobertura de idiomas (latín, cirílico, griego, CJK, árabe, hebreo, thai y más).

Instalación

Recomendado — .cia vía Universal-Updater:

  1. Instala 3dslibris desde Universal-Updater.
  2. Ábrelo una vez para crear sdmc:/3ds/3dslibris/ si es necesario.
  3. Copia tus libros a sdmc:/3ds/3dslibris/book/.

Manual — .3dsx:

  1. Descarga 3dslibris-sdmc.zip y 3dslibris.3dsx desde GitHub Releases.
  2. Extrae el zip en la raíz de la SD.
  3. Copia 3dslibris.3dsx a sdmc:/3ds/3dslibris/3dslibris.3dsx.
  4. Copia tus libros a sdmc:/3ds/3dslibris/book/.
